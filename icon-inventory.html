Peek Icon Inventory

Visual reference for all icons used in the Peek application. Generated from codebase audit.

Audit Summary

  • Fixed: Studio indicator now uses Building2 (was Clapperboard)
  • Fixed: Favorite Studios carousel now uses Building2 entity icon (was Video)
  • Fixed: Favorite Tags carousel now uses centralized Tag entity icon
  • Fixed: RatingCell now uses RatingBadge (bronze/silver/gold) instead of 5-star display
  • Fixed: Removed dead LucideStar imports from PerformerDetail, StudioDetail, TagDetail
  • Fixed: TagTreeNode favorite indicator now uses Heart (was Star) for consistency
  • Fixed: FilterPresets "default preset" indicator now uses Pin (was Star) for clarity
  • Note: Star used for "High Rated" carousel - conceptual icon for quality, acceptable
